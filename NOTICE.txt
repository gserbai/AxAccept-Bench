3. Ramulator
   Source: https://github.com/CMU-SAFARI/ramulator
   License: MIT License

   Description:
   A cycle-accurate DRAM simulator used to generate DRAM command traces from
   filtered AxRAM memory traces.

   AxAccept-Bench uses Ramulator with the following DDR3 configuration:

   standard = DDR3
   speed    = DDR3_1600K
   org      = DDR3_2Gb_x16

---

4. DRAMPower 4.1
   Source: https://github.com/tukl-msd/DRAMPower
   License: BSD 3-Clause License

   Description:
   A DRAM power and energy estimation tool used to estimate energy from
   Ramulator command traces.

   AxAccept-Bench uses the following DRAMPower memory specification:

   DRAMPower-4.1/memspecs/MICRON_2Gb_DDR3-1600_16bit_D.xml
